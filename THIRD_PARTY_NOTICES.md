## Stockfish

Stockfish is a separate chess-engine executable distributed under GPLv3.
It is not original ChessTeacher code. A release that bundles Stockfish must
also bundle the exact upstream copyright/licence material and provide the
corresponding source or a compliant written/source-location offer.

The included download and release workflow preserves upstream licence files
next to the engine whenever those files are present in the official archive.

## Chess symbols and icon

The first board renders standard Unicode chess symbols using the user's system
font. The application icon in this project is an original generated asset.
No Chess.com branding, logos, private algorithms, or proprietary assets are used.
